# Follow The Message

In deze repo vind je de slides voor de workshop Follow the Message die we vanuit de opleiding bachelor toegepaste informatica aanbieden aan leerlingen van het secundair onderwijs.

## Cisco PacketTracer

In deze workshop maken we gebruik van [Cisco PacketTracer](https://www.netacad.com/courses/packet-tracer), een simulator voor computernetwerken. PacketTracer kan gratis gedownload en gebruikt worden, maar je hebt een (eveneens gratis) account nodig op Cisco Netacad, <https://www.netacad.com/>.

## Labo-opgave

Uitgewerkte instructies voor de labo-opgave vind je [hier](labo.md). PacketTracer-bestanden met de begin- en eindsituatie vind je in de directory `packettracer/`. Deze zijn aangemaakt in PacketTracer 7.1.1. Als je zelf met een nieuwere versie werkt, is het mogelijk dat deze niet correct kunnen geopend worden.

## Slides

De slides zijn opgemaakt met LaTeX. Informatie/handleiding over het gebruikte sjabloon vind je hier: <https://github.com/HoGentTIN/presentatie-latex-sjabloon>. Je kan een pdf-versie downloaden als je hierboven doorklikt op "releases".
